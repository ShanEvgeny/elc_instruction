1

Добавление файла

На странице курса нажмите «Режим редактирования».

Нажмите «Добавить элемент или ресурс» в одном из разделов курса.

В открывшемся окне выберите элемент Файл.

2

Настройки файла

Общее: заполните обязательное поле «Название» и при необходимости «Описание».

Выбор файла: перетащите файл в область загрузки или нажмите «Добавить» для выбора из репозитория.

3

Параметры отображения

В разделе «Внешний вид» выберите способ отображения файла:

  • Автоматически — Moodle определит оптимальный способ
  • Встроить — файл откроется внутри курса (для PDF, изображений)
  • Открыть — файл откроется в новом окне
  • Скачать — файл будет автоматически скачан
💡 Если файл не отображается в интерфейсе, студентам будет предложено скачать его

Нажмите «Сохранить и вернуться к курсу».

Файл добавлен! Студенты могут просматривать или скачивать ресурс.

✅ При клике на файл происходит автоматическое скачивание или открытие в зависимости от настроек
