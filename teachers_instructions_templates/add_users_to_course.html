1

Зачисление студентов на курс по группам

Для зачисления группы на главной странице курса, в блоке «Настройки» выберите «Пользователи» → «Способы зачисления на курс».

Затем в раскрывающемся списке «Добавить способ» выберите «Зачислить группу (все специальности)».

Далее укажите необходимую группу и нажмите «Добавить способ».

Все студенты выбранной группы будут зачислены на ваш курс и увидят его на главной странице сайта.

2

Индивидуальное зачисление

На главной странице курса, в блоке «Настройки» нажмите «Пользователи» → «Зачисленные на курс пользователи».

В правом верхнем углу страницы Участники нажмите Зачисление пользователей на курс

На следующем окне в поле Выберите пользователей введите либо фамилию, либо имя, отчество, либо сначала имя, отчество, а потом фамилию (например, для тех случаев, когда слишком много пользователей с одинаковыми именами). Другие комбинации не будут найдены.
Затем нажмите на необходимого пользователя(-ей), чтобы он закрепился полем выше.
После этого нажмите «Зачислить выбранных пользователей и глобальные группы».

Если в списке появляются полные тезки, обратите внимание на поле рядом — это группа, в которой числится пользователь. Если Вам нужно зачислить преподавателя или тьютора и т.п. на курс, измените роль в соответствующем поле при зачислении.
3

Самостоятельная запись на курс

Для зачисления группы на главной странице курса, в блоке «Настройки» выберите «Пользователи» → «Способы зачисления на курс».

Затем в раскрывающемся списке «Добавить способ» выберите «Самостоятельная запись».

Обязательно введите Название способа и кодовое слово (его не нужно никому сообщать), оно требуется для работы данного способа записи. В поле «Использовать кодовые слова для групп» отметьте «Да». Нажмите «Добавить способ».

В блоке «Настройки» выберите «Управление курсом» → «Пользователи» → «Группы».

Подобных групп можно создать неограниченное количество, главное чтобы кодовые слова были уникальными.

Нажмите «Создать группу», введите название группы и кодовое слово (его нужно будет сообщить студентам), щелкните «Сохранить».

5

Демонстрация

Студент одним из доступных ему способов находит курс, на который ему необходимо зачислиться (например, для сдачи долга).

Далее указывает кодовое слово (установленное для группы), нажимает Записаться на курс, и автоматически студент будет зачислен в нужную группу на курсе.
